      Address: Enter the first part of the address. Ex: for 700 North Tryon Street, try 700 n

      Place or Street: Enter any part of a place or street name. Ex: type carolina

      Intersection: Enter a street name, followed by an &. Ex: type ruth &

      Parcel ID: Enter an 8-digit Parcel ID. Ex: type 11111111

      Find My Location: If your web browser supports geolocation, you will see Find My Location under the search bar. Your browser will ask you for permission to use this feature the first time it is activated.

      • Routing, including driving, walking, biking, and transit, is now built-in!
      • Totally refactored code base for even faster performance.

      • You might notice the hash (#XXXXX) that appears in the URL bar when a property is selected. This enables your browser history, so if you select multiple records you can go forward and backward in your history. You can also email the URL when a record is selected and the recipient will automatically select the record!
      • Install the Google Earth API to experience GeoPortal in 3D! After installation, an Earth option will appear in the list on top right of the map.
